# Visual Directions — Theme Mockups

Two visual directions are implemented as toggleable themes in the app (`Theme` button top-right). Switch between them to preview.

1) Glassy / Soft (default)
- Palette: soft indigo to pink gradient background, translucent glass cards.
- Feel: calm, professional, minimal; emphasis on readable typography and subtle depth.
- Key UI features: blurred glass panels, soft drop shadows, smooth micro-interactions.

2) Vibrant / Illustrative
- Palette: warm peach to coral gradient background, slightly more saturated accents.
- Feel: energetic, friendly, modern; better for younger audiences and playful presentation.
- Key UI features: warmer shadows, brighter CTA gradients, stronger highlights on top results.

How to preview
1. Run the app: `npm run dev`
2. Use the `Theme` toggle in the app header to switch mockups.

Notes
- Both themes use the same layout and scoring logic — only visuals change.
- If you want image artwork or custom illustrations for the Vibrant direction, I can add an SVG hero and accent shapes.
